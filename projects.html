Flappy Xina Game

Flappy Xina is a game developed using C# for the Unity engine, inspired by the classic "Flappy Bird" gameplay. In this version, players control a character named Xina, navigating through obstacles and aiming for a high score. The game was built to explore physics mechanics and game development in Unity.
